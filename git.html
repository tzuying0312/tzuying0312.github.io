新增/還原/忽略

  • 帳戶設定
    • git config --global user.name "your name" 設定姓名
    • git config --global user.email "your email" 設定電子郵件
  • 查詢資訊
    • git version 查詢版本
    • git config --list 查詢設定列表

  • 增加檔案
    • git add your_filename 增加單一檔案進入索引
    • git add . 增加全部檔案進入索引
    • git status 查詢狀態
    • git commit -m '更新訊息' 將索引提交到數據庫
  • 忽略
    • 新建 .gitignore 檔案，裡面存入要忽略的檔案
    • index.html 忽略 index.html 檔案
    • *.html 忽略全部 HTML 檔案
    • css/ 針對 css 資料夾忽略

  • 還原指令
    • git reset --hard 還原工作目錄與索引，會跟最後一次 commit 保持一樣
    • git checkout HEAD -- db.sqlite3 同上，但為還原單一檔案
    • git reset HEAD 全部檔案取消索引
    • git reset HEAD your_filename 同上，但為取消單一檔案索引
    • git checkout your_filename 恢復單一檔案到最新 commit 狀態
  • 還原指令(續)
    • git reset --hard HEAD^ 刪除最近一次 commit
    • git reset --hard ORIG_HEAD 上面語法如果刪除錯了可以再用此語法還原
    • git reset --soft HEAD^ 刪除最近一次 commit，但保留異動內容
    • git rm -r --cached . 全部變成未追蹤
    • rm -rf .git 刪除 git

遠端數據庫

  • 遠端數據庫
    • git clone url 複製遠端數據庫
    • git remote 查詢遠端數據庫
    • git remote -v 查詢詳細遠端數據庫
    • git remote 遠端數據庫名 遠端數據庫 刪除 Git 遠端倉庫的 URL
    • git remote rm 遠端數據庫名 切換分支
    • git remote rename 原名稱 新名稱 修改遠端數據庫名稱
    • git push 遠端數據庫名稱 遠端分支名稱 將本地分支推送到遠端分支
    • git pull 將遠端分支拉下來與本地分支進行合併
    • heroku git:remote -a app_name 新增 heroku remote
    • git push remote_name -d branch_name 刪除遠端分支
    • git push -f 強制推送
    • git push heroku branch_name:master 上傳 heroku 分支

分支/標籤/暫存

  • 分支
    • git branch 查詢所有分支
    • git branch -a 查詢詳細所有分支
    • git branch 分支名稱 新增分支
    • git checkout 前四碼 將 HEAD 指向該 commit 還原到該 commit
    • git checkout 分支名稱 切換分支
    • git merge 分支名稱 合併指定分支到目前的分支
    • git branch -d 分支名稱 刪除分支
    • 註:可透過 git log 查看 commit 前四碼

  • 暫存
    • git stash 暫時儲存當前目錄
    • git stash list 瀏覽 stash 列表
    • git stash pop 還原暫存
    • git stash drop 清除最新暫存
    • git stash clear 清除全部暫存
  • 標籤
    • git tag 查詢標籤
    • git tag -n 查詢詳細標籤
    • git tag -d 標籤名稱 刪除標籤
    • git checkout 標籤名稱 切換到標籤的 commit
